PROBLEMA N-QUEENS
PATH: "framework\problems\iterative_improvement_problem"

per eseguire il problema N-Queens, ci sono 2 modi:
     1. ./main <algoritmo> N
     2. make run   (esegue tutti gli algoritmi con N=4: ./main All 4)  
     
algoritmi disponibili: Hill | Steepest | Simulation_Annealing | All



PROBLEMI DI RICERCA DEGLI STATI
PATH: "framework\problems\search_state_problem"

     1. ./main <algoritmo> <problema>
